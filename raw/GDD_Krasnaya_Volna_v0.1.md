# «КРАСНАЯ ВОЛНА» — GDD-скелет v0.1

> Рабочее название игры: **«Красная Волна» (Red Wave)**
> Жанр: кооперативная тайм-менеджмент-игра про отдых в аквапарке
> Движок: Godot 4.x, low-poly 3D + toon-шейдинг
> Игроки: 1–4, host-listen кооператив
> Длительность забега: 30 минут реального времени = один день в парке

---

## 1. Питч

Один день в гигантском аквапарке «Красная Волна» — месте, где хорошо всем без исключения. Утром парк пуст и принадлежит только вам; к полудню он наполняется толпой, очередями и гулом. У вас есть список желаний на день и неумолимые часы. Успейте всё — или просто лягте на ленивую реку и плывите. Вечером — фейерверк для всех.

**Эмоциональная цель:** телесное удовольствие воды + ностальгия по «дню в парке» + лёгкая тревога тайм-менеджмента + ощущение, что парк больше и старше, чем кажется.

**Референсы:** Парк Паэлья (правила), «Унесённые призраками» (свет, купальня, масштаб), «Мор. Утопия» (театр, недосказанность), A Short Hike (свобода и доброжелательность мира).

## 2. Структура дня (30 минут реального времени)

Игровой день: 09:00 → 21:00. Курс: **1 игровой час = 2,5 минуты реального времени.**

| Фаза | Реальное время | Игровое время | Состояние парка |
|---|---|---|---|
| Утро | 0:00–7:30 | 09:00–12:00 | Пусто. Очередей нет. Свет холодный, голубой, длинные тени. Звук: эхо, капли, одинокие шаги. |
| Полдень | 7:30–17:30 | 12:00–19:00 | Пик. Очереди 60–180 сек. Свет тёплый, плотный. Гул толпы, музыка из динамиков. |
| Вечер | 17:30–27:00 | 19:00–20:30 | Спад. Очереди 30–60 сек. Закат, янтарный свет, фонари. |
| Финал | 27:00–30:00 | 20:30–21:00 | Все стягиваются к Приливу. Фейерверк. Подсчёт очков. |

**Расписание событий (фиксированное, известно игрокам с начала):**

| Игровое время | Реальное время | Событие |
|---|---|---|
| 11:00 | 5:00 | Утреннее представление в театре «Рупор» (90 сек) |
| 13:00 | 10:00 | Парад маскотов по главной аллее (60 сек, фотоокно) |
| 15:00 | 15:00 | Случайное закрытие одной горки на техобслуживание (2–4 мин) |
| 17:00 | 20:00 | Вечернее представление в театре «Рупор» (120 сек) |
| 20:30 | 27:00 | Фейерверк над Приливом (финал) |

Опоздал к началу представления → двери закрыты, квест провален. Никаких поблажек: это главный источник драмы планирования.

## 3. Карта парка

Парк — кольцо. В центре Большая Купальня, вокруг неё кольцом течёт ленивая река «Капилляр», внешнее кольцо — горки и зоны. Река — одновременно транспорт (плыть по течению между зонами) и отдых.

```
                    [Ротонда / Вход]
                          |
   [Лавки + Фудкорт «Жажда»]   [Театр «Рупор»]
            \                       /
   [Горка «Клык»]   ===========   [Горка «Хоботок»]
            \      ( КАПИЛЛЯР )      /
             \    (  кольцевая  )   /
   [Воронка]  --- [ БОЛЬШАЯ     ] --- [Волновой
              --- [ КУПАЛЬНЯ    ] ---  бассейн
             /    (   река      )   \  «Прилив»]
            /      ===========       \
   [«Лужица» детская]          [Терраса «Отмель»]
```

### 3.1 Локации

**Ротонда (вход).** Хаб. Высокий атриум с витражным куполом — первый кадр игры, «вау» купальни из «Унесённых призраками». Здесь: выдача квестов, карта парка, шкафчики (точка респауна), большие часы — всегда видны.

**Большая Купальня (сердце парка).** Тёплый зал с десятками ванн разной температуры, пар, латунь, дерево, тёплый ламповый свет. Геймплейно: зона полного сброса головокружения, место общих сборов, медленная зона (бегать нельзя — поскользнёшься, анимация падения). Атмосферный якорь всей игры.

**Ленивая река «Капилляр».** Кольцевая, течёт по часовой стрелке, полный круг — 90 сек. Транспорт + отдых (сбрасывает головокружение). Можно болтать, держась за один круг вдвоём (кооп-анимация).

**Горка «Клык».** Экстремальная, почти вертикальный спуск с башни. Самая высокая точка парка — с башни видно весь парк (и кое-что за его стенами). Головокружение +3. Самые длинные очереди днём.

**Горка «Хоботок».** Длинная спиральная закрытая труба с просветами. Головокружение +2. Внутри трубы — секции с разноцветной подсветкой (дёшево, эффектно).

**«Воронка».** Гигантская воронка-улитка: разгон, кружение, выплёвывает в бассейн. Головокружение +2. Катается до 4 игроков одновременно на одном круге — главный кооп-аттракцион.

**«Лужица».** Пологая семейная горка и лягушатник. Головокружение 0, очередей почти нет. Зона для чилла и пары «тихих» квестов.

**Волновой бассейн «Прилив».** Большая чаша с волнами по расписанию (волна каждые 40 сек реального времени). Мини-игра «оседлай волну». Место финального фейерверка.

**Театр «Рупор».** Деревянный амфитеатр под навесом. Представления по расписанию: куклы и маски, стиль «Мор. Утопии». Спектакли — короткие притчи о воде, жажде и о том, кто построил парк. 2–3 разных спектакля в ротации (реиграбельность + нарративный слой).

**Фудкорт «Жажда» + лавки (3 шт.).** Еда (сброс головокружения, квесты «попробуй фирменный коктейль»), сувенирная лавка, лавка купального (косметика: круги, полотенца, очки), лавка масок (косметика + связь с театром).

**Терраса «Отмель».** Шезлонги, зонты, вид на Прилив. Пассивный сброс головокружения. Точка «ничегонеделания» для мирного режима.

**Маскот «Бард».** Блуждающий NPC в нелепом костюме — никто никогда не видел его на одном месте дважды. Аналог неуловимой Королевы зомби из Паэльи: квест «сфотографируйся с Бардом» — лотерея и азарт. По парку ходят слухи, существует ли он вообще.

## 4. Таблица временных стоимостей

Все значения — реальные секунды. Базовая валюта игры — время.

### 4.1 Перемещение

| Действие | Стоимость |
|---|---|
| Пешком между соседними зонами | 15–25 сек |
| Пешком через весь парк | 60–80 сек |
| По реке «Капилляр» (по течению, между соседними зонами) | 20–30 сек (но руки заняты, нельзя есть) |
| Бег (вне Купальни) | ×0,7 ко времени, но +шанс поскользнуться у бассейнов (падение = 5 сек + смешно) |

### 4.2 Аттракционы

| Горка | Очередь утром | Очередь днём | Очередь вечером | Сам спуск | Головокружение |
|---|---|---|---|---|---|
| «Клык» | 0 | 150–180 сек | 60 сек | 25 сек | +3 |
| «Хоботок» | 0 | 90–120 сек | 45 сек | 35 сек | +2 |
| «Воронка» | 0 | 90 сек | 30 сек | 30 сек | +2 |
| «Лужица» | 0 | 20 сек | 0 | 15 сек | 0 |
| «Прилив» (волна) | без очереди | без очереди | без очереди | 40 сек цикл | +1 за 3 волны |
| «Капилляр» (круг) | 0 | 15 сек на вход | 0 | 90 сек круг | −2 (сбрасывает!) |

### 4.3 Прочие действия

| Действие | Стоимость | Эффект |
|---|---|---|
| Поесть на фудкорте | 25 сек | Головокружение −2 |
| Купить сувенир / предмет | 15 сек | Предмет, прогресс квестов |
| Сфотографироваться (точка) | 10 сек | Прогресс квестов |
| Фото с Бардом | 10 сек + найти его | Квест-лотерея |
| Представление в театре | 90–120 сек | Только по расписанию |
| Полежать на «Отмели» | пассивно | Головокружение −1 / 15 сек |
| Ванны Купальни | пассивно | Головокружение −1 / 8 сек |
| Шкафчик (сменить косметику) | 10 сек | — |

### 4.4 Риск: пройти без очереди (колесо Паэльи → QTE)

| Параметр | Значение |
|---|---|
| Попытка | мини-QTE 8 сек (проскользнуть мимо охраны, тайминг-полоска) |
| Успех | сразу на горку, экономия = вся очередь |
| Провал | охранник ведёт в «комнату бесед» = 90 сек штрафа + жетон позора |
| 2 жетона позора | охрана узнаёт в лицо: QTE усложняется до конца дня |

### 4.5 Головокружение

Шкала 0–5. При 5: экран плывёт, скорость ×0,6, персонаж шатается, нельзя на горки с +2/+3 (контролёр не пускает: «вам бы посидеть»). Сброс: еда, Капилляр, Отмель, Купальня (см. выше).

## 5. Квесты

В начале забега каждый игрок тянет **2 личных квеста** (закрытая информация — соревновательный нерв) + команда получает **1–2 общих** (видны всем). Подбор с проверкой на конфликты расписания (нельзя выдать два квеста на одно и то же время).

### 5.1 Личные квесты — стартовый пул (20 шт.)

**Горки:**
1. Скатись с «Клыка» до полудня (пока нет очередей)
2. Скатись со всех четырёх горок за день
3. Скатись с «Хоботка» три раза
4. Скатись с «Клыка» на закате (после 19:00)
5. Прокатись по «Воронке» в полной кабинке (4 человека — личный квест, требующий кооперации!)
6. Поймай 5 волн подряд в «Приливе»

**Время и риск:**
7. Проскочи без очереди дважды и не попадись
8. Проведи в очередях суммарно меньше 60 секунд за день
9. Сделай полный круг по «Капилляру», ни разу не выйдя (3 круга подряд)
10. Доведи головокружение до 5 и полностью сбрось его до вечера

**Места и события:**
11. Посети оба представления в «Рупоре»
12. Сфотографируйся с Бардом
13. Сфотографируйся на вершине башни «Клыка»
14. Попади на парад маскотов (13:00) и сделай фото
15. Полежи в трёх разных ваннах Большой Купальни

**Покупки и еда:**
16. Попробуй три разных блюда «Жажды»
17. Купи маску в лавке масок и не снимай её до фейерверка
18. Купи сувенир каждому члену команды (кооп-щедрость)
19. Потрать все карманные деньги до 18:00
20. Съешь мороженое на вершине «Клыка» (купить внизу, донести, не уронить в очереди)

### 5.2 Общие квесты — стартовый пул (8 шт.)

1. **Полный круг.** Вся команда на одном круге «Воронки» одновременно
2. **Премьера.** Вся команда на вечернем представлении (17:00) — на соседних местах
3. **Цепочка.** Вся команда плывёт по «Капилляру» одним паровозиком (держась за круги) полный круг
4. **Синхрон.** Двое скатываются с «Клыка» и «Хоботка» одновременно (старт в одну секунду)
5. **Большая волна.** Вся команда ловит одну и ту же волну в «Приливе»
6. **Банный день.** Вся команда одновременно в одной ванне Купальни минимум 20 сек
7. **Охота на Барда.** Хотя бы трое успевают сфотографироваться с Бардом за день
8. **Все на фейерверк.** Вся команда на террасе «Отмель» к 20:30 (квест-страховка: учит приходить к финалу)

### 5.3 Подсчёт очков (соревновательный режим)

| Достижение | Очки |
|---|---|
| Личный квест | 5 |
| Общий квест | 4 каждому |
| Каждая уникальная горка за день | +1 |
| Жетон позора | −2 |
| Не успел к фейерверку | −3 |
| Бонус «без единой очереди скучания» (ни разу не стоял 120+ сек) | +2 |

Побеждает игрок с максимумом очков, но общие квесты делают победу «тёплой»: выгодно помогать.

## 6. Режимы

**Соревновательный забег (30 мин).** Всё описанное выше. 1–4 игрока.

**Мирный режим (без таймера).** Та же симуляция парка, день длится бесконечно (цикл фаз повторяется). Квестов нет, очков нет. Доступно: горки, мини-игры, шопинг, театр по расписанию, фотомод. Это «Неверленд»: место, куда заходят просто побыть.

---

## 7. Водные мини-игры (фаза 4, краткие концепты)

1. **Брызгалки.** Арена в «Лужице», водяные пистолеты, 2×2 или каждый за себя, 90 сек
2. **Оседлай волну.** «Прилив»: удержаться на круге через серию волн нарастающей силы
3. **Ныряние за монетами.** Дно «Прилива» между волнами: собери больше монет на одном вдохе (шкала воздуха)

---

## 8. Нарративный слой (фоновый, ненавязчивый)

Парк старше, чем выглядит. В театре «Рупор» играют притчи, в которых внимательный зритель заметит: парк построен поверх чего-то — труб, каналов, инфраструктуры, назначение которой никто из персонажей не обсуждает. Бард, возможно, помнит. Никаких прямых ответов: только спектакли, обмолвки NPC в очередях и вид с башни «Клыка». Игра остаётся весёлой; тень — на периферии зрения, как и положено палимпсесту.

## 9. Технические рамки (фиксируем решения)

- Godot 4.x, GDScript, git с первого коммита
- Low-poly 3D, toon-шейдер + рамп-освещение, стилизованная вода (шейдер: вертексные волны + френель + caustics-текстура)
- Камера: третье лицо, мягкая орбитальная, авто-кадрирование на горках (rail-камера во время спуска)
- Сеть: high-level multiplayer API, host-listen, авторитет сервера на квестах и часах
- Персонажи: единый риг, кастомизация цветом и аксессуарами (дёшево)
- NPC-толпа: до 100 болванчиков, упрощённый LOD, навигация NavMesh, спавн по расписанию фаз дня
