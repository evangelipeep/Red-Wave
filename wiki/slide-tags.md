# Теги горок

Каждая из 15 горок ([[park-map]]) описана набором осей-тегов. Теги — основа
генератора квестов ([[quest-system]]): атомы вроде «прокатись на 3 разных
ощущениях» опираются именно на них.

## Содержание

Модель горки:
`Горка { id, зона, головокружение 0..3, тип, ощущение, температура, гул 0..100, состояние, экстрим, спокойная }`

**Оси-теги:**
- **Ощущение (6):** падение · мрак · скольжение · невесомость · погружение ·
  кружение. Каждое представлено минимум в 2 зонах.
- **Температура (отдельная ось, 7-й слой):** ледяная · тёплая · нейтральная.
- **Тип:** башня, труба, жёлоб, спуск, лодки, гонка, воронка, капсула, спираль и т.п.
- **Головокружение:** 0..3 — насколько укачивает (питает [[nausea-system|тошноту]]).
- **Экстрим:** по одному на зону (Клык / Жало / Ва-банк); требует веса ≤90
  ([[weight-system]]).
- **Спокойная:** по одной на зону, все «лодки», ≈60 сек — медитативные, питают
  «тихие квесты».

Покрытие тегов спроектировано так, чтобы каждое ощущение встречалось ≥2 зонах, а
крайние температуры — у нескольких горок; это даёт генератору достаточно
материала для сбалансированных бандлов.

## Связано с

- [[park-map]] — полные таблицы горок по зонам
- [[quest-system]] — атомы квестов опираются на теги
- [[weight-system]] — лок экстрима по весу
- [[nausea-system]] — головокружение горки → тошнота
