# Подсчёт очков

Очки делятся на «назначенное» (равный базовый счёт за выполненные квесты) и
«динамическое» (создаёт разрыв, равнодоступно всем). Разрыв между игроками рождает
только динамика — назначенное у всех выровнено по двум осям.

## Содержание

**Назначенное (равный базовый счёт):**
- главный квест — 20 (по 20/K за подзадачу),
- личные ×2 — по 5,
- общие — по 4.

**Динамическое (разрыв, равнодоступно):**

| Источник | Очки |
|---|---|
| [[zone-pioneer-bonus|Первопроходец зоны]] ×3 | +5 каждая |
| Победа в [[roy-race|«Рое»]] | +3 (макс +9) |
| [[side-missions|Доп-миссия]] | +5 / −3 |
| Уникальная горка | +1 (макс +15) |
| Бережливый бонус ([[economy-coins]]) | +floor(монеты/2), макс +5 |
| Ни одной очереди >120с | +2 |
| Жетон позора (хулиган, [[anti-exploit]]) | −2 |
| Не в Капилляре к [[finale-ballad|Балладе]] | −3 |

**Сложный режим** ([[difficulty-presets]]) добавляет штрафы за невыполненные
назначенные квесты.

**Тай-брейк:** больше квестов → больше зон → меньше очередей → выше бережливость.
