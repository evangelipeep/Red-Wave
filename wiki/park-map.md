# Карта парка

Парк = три тематические зоны по 5 горок + центральный хаб. Каждая горка описана
тегами (см. [[slide-tags]]). По одному «экстриму» и одной «спокойной» на зону.

## Содержание

### Зона «Северный Клык» (вампирская, готика, полумрак)

| Горка | Гол. | Тип | Ощущение | Темп. | Тег |
|---|---|---|---|---|---|
| Клык — вертикаль с башни | +3 | башня | падение | нейтр. | экстрим |
| Плащ — тёмная труба | +2 | труба, темнота | мрак | ледяная | |
| Крыло — открытый жёлоб с виражами | +2 | жёлоб | невесомость | нейтр. | |
| Витраж — труба со светом, нырок | +1 | труба | погружение | тёплая | |
| Колыбельная — лодки-гробики | 0 | лодки | скольжение | ледяная | спокойная |

### Зона «Дельта» (комариная, болото, шумная)

| Горка | Гол. | Тип | Ощущение | Темп. | Тег |
|---|---|---|---|---|---|
| Жало — спуск с трамплином | +3 | спуск, полёт | невесомость | тёплая | экстрим |
| Хоботок — спираль с подсветкой | +2 | труба, спираль | кружение | нейтр. | |
| Топь — спуск с всплеском | +2 | спуск | погружение | тёплая | |
| Рой — гонка на 4 дорожки | +1 | гонка | скольжение | нейтр. | |
| Мангры — тёмный тоннель-лодка | 0 | лодки, темнота | мрак | тёплая | спокойная |

### Зона «Серый Пояс „Зеро"» (казино, неон, случайность)

| Горка | Гол. | Тип | Ощущение | Темп. | Тег |
|---|---|---|---|---|---|
| Ва-банк — trapdoor-старт | +3 | капсула | падение | нейтр. | экстрим |
| Рулетка — воронка, до 4 чел. | +2 | воронка, групповая | кружение | нейтр. | |
| Джекпот — три случайных выхода | +2 | труба, рандом | погружение | нейтр. | |
| Размен — раздвоение на 2 дорожки | +1 | труба, рандом | скольжение | ледяная | |
| Ноль — медленный неоновый тоннель | 0 | лодки, темнота | мрак | ледяная | спокойная |

### Центр

Ротонда (хаб, табло Гула, Стойка информации, шкафчики, часы, весы), Большая
Купальня, река «Капилляр» (кольцо с течением), «Прилив», «Лужица», «Отмель»,
фудкорт «Жажда» ([[food-court]]), центральный туалет, маскот [[bard-mascot|Бард]].
Горка «Рой» — это гонка, см. [[roy-race]].

> Реализация горок — `SlideRail` (труба `CSGPolygon3D` вдоль `Path3D`), см.
> [[godot-architecture]] и [[art-style]].
