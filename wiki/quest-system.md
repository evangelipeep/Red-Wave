# Система квестов (генератор)

Желания дня выдаются генеративной формулой из библиотеки атомов-предикатов. Главный
квест собирается так, чтобы баланс был по ДВУМ осям — сложности и времени — что
проверено симуляцией. Если генератор не справился — берётся [[fallback-bundles|резервный бандл]].

## Содержание

**Раздача (сбалансированная):**

| Слот | Кол-во | Очки |
|---|---|---|
| Главный квест-формула | 1 | 20 (по 20/K за подзадачу), бандл 2–3 атома |
| Личные | 2 | 5 каждый, один атом |
| Общие (кооп) | 1–2 | 4 каждому, одинаковы для всех |

Все назначенные выровнены по двум осям → если каждый выполнит назначенное, базовый
счёт равный. Разрыв создают только динамические бонусы ([[scoring]]).

**Библиотека атомов** (примеры): катание — `RIDE_DISTINCT`, `RIDE_EXTREME`,
`RIDE_SENSATION(s,n)`, `RIDE_TEMP`, `RIDE_GUL_BELOW`, `ONE_PER_ZONE`, `CLOSE_ZONE`,
`RIDE_CALM`, `SUNSET_RIDE`, `RACE_WIN`; события — `ATTEND_SHOWS`, `PHOTO_BARD`,
`PHOTO_PARADE`; тело/экономика — `SPEND_ALL_BY`, `FOOD_ALL_ZONES`,
`WEIGHT_END_BELOW`, `WEIGHT_EXTREME_AT`, `BATHS`; риск — `QUEUE_SKIP`,
`CAPILLYAR_LAPS`. Метаданные атома: `ep`, `t_est`, `coin_cost`, `axis`, `min_supply`.

**Две оси баланса (ВАЛИДИРОВАНО):** симуляция 600 забегов показала, что баланс
только по сложности (`ep=14`) давал разброс времени 19–58% дня — «равная сложность»
врала. Добавление коридора времени 30–50% дня схлопнуло разброс (σ≈5.4 п.п., 0
провалов на 300 забегов).

```
D_target = 14 ep (±2)        # ось 1: сложность
T_band   = [30%, 50%] дня    # ось 2: время (с учётом Гула)
payout   = 20 (фикс)
собрать bundle (2–3 атома) через COMPATIBLE();
принять, если Σep ∈ [12,16] И Σt_est ∈ T_band; иначе откат (лимит 60);
fallback: безопасный список провалидированных бандлов (баг #11).
```

**COMPATIBLE()** — не соединять провальное: коллизия фиксированного времени (макс
1 жёстко-временной атом), противоречия тела, бюджет времени (главный ≤50% дня, все
назначенные ≤85%), бюджет монет ≤10, достаточность объектов, конфликт с
техобслуживанием, дедуп осей, разделение соло/кооп.

**SOLVABLE()** — симуляция маршрута: события по времени → гибкие атомы в промежутки
→ `t += дорога(NavMesh) + очередь(Гул,фаза) + действие` с детурами (туалет/остывание)
→ итог ≤ день×0.85.

**Тихие квесты** (`RIDE_CALM`, `BATHS`, `CAPILLYAR_LAPS`, `RIDE_SENSATION(мрак)`) —
медитативные, дают честные очки за «потерю времени», балансируют гонку за экстримом.

**Общие (кооп) квесты:** Полная «Рулетка», Премьера (17:00), Цепочка-паровозик,
Синхрон (два экстрима, окно 2с), Большая волна, Банный день, Охота на [[bard-mascot|Барда]]
(≥3), Фотофиниш (вся команда в [[roy-race|«Рое»]]), Все на [[finale-ballad|Балладу]],
«Семь стихий» (все ощущения + обе крайние температуры).
