# Аудио

Сюда кладём звуки. Форматы: **.wav** (короткие SFX — брызги, выстрел), **.ogg** (музыка,
длинные/зацикленные). `.mp3` тоже работает.

## Как добавить звук, чтобы он заработал
1. Скопируй файл в эту папку (`res://assets/audio/…`) — Godot сам импортирует.
2. Выдели файл в доке **FileSystem** → вкладка **Import**: для коротких SFX выключи Loop,
   для музыки включи Loop → **Reimport**.
3. Назначь звук туда, где он играет (см. ниже).

## Куда что назначать
- **Брызги воды** — экспорт `Splash Sound` на узле **Player**
  (`player/PlayerController.gd`). Перетащи сюда `splash.wav`.
  Двойной всплеск для тяжёлого игрока (≥85 кг) — уже автоматический
  (`GameConstants.SPLASH_HEAVY_KG` / `SPLASH_HEAVY_DELAY`).
- **Выстрел пистолета-отталкивателя** — экспорт `Gun Sound` на узле **Player**.
  Перетащи `gun.wav`. Без него стрельба просто беззвучная.
- **Эмбиент/прочее** — добавляй `AudioStreamPlayer3D` рядом с объектом и ставь `stream`.

Громкость брызг крутится в `_play_splash_sound()` (volume_db), задержка двойного — в
`GameConstants.SPLASH_HEAVY_DELAY`.
