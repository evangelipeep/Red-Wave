# UI-картинки

## Ячейка инвентаря (рамка слота) — переиспользуемая
Положи сюда:
- **`slot.png`** — обычная (пустая) рамка ячейки;
- **`slot_selected.png`** — рамка выбранной ячейки (подсветка).

Как только файлы появятся — хотбар (и будущие сундуки) автоматически возьмут их
(`Look.slot_style()` в [autoload/Look.gd](../../autoload/Look.gd)). Нет файлов — рисуется
плоский фолбэк, ничего не ломается.

**Один источник на всё:** и хотбар, и сундуки зовут `Look.slot_style(selected)`. Поэтому
твоя ячейка — действительно переиспользуемая: загрузил один раз, применяется везде.

### Требования к картинке
- Квадратная, например **96×96 px** (можно 64–128).
- Это **9-патч**: рамка по краям, центр пустой/полупрозрачный (под иконку). Поля
  9-патча сейчас 18 px со всех сторон — правится в `Look._make_slot` (`texture_margin_*`),
  подгони под толщину своей рамки.
- Формат **.png** с прозрачностью.

## Иконки предметов (по желанию, позже)
Иконки в ячейках (стакан/круг/зонт/мышь) сейчас — эмодзи. Если захочешь картинки:
клади `assets/ui/items/<id>.png` (напр. `pill.png`, `gun.png`, `coffee.png`) — подключим
их в хотбар вместо эмодзи (скажи, сделаю). Размер ~64×64, прозрачный фон.
