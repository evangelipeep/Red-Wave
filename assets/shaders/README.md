# Шейдеры — тун-стиль (Wind Waker)

Полный гайд по стилю: [../../ART_STYLE.md](../../ART_STYLE.md).
Здесь — кратко про сами файлы шейдеров.

## Файлы

| Файл | Что делает |
|---|---|
| `toon.gdshader` | **Непрозрачный** тун-материал: плоский цвет + ступенчатый свет. Основной (95% геометрии). |
| `toon_alpha.gdshader` | **Прозрачный** тун: то же освещение + альфа (стекло, вода, подсветки зон). |
| `toon_light.gdshaderinc` | Общая функция ступенчатого света. Подключается `#include` в оба шейдера выше — формула в одном месте. |
| `outline.gdshader` | Чёрный контур методом «вывернутого корпуса». Вешается как `next_pass`. |

## Как это применяется

Руками шейдеры вешать **не нужно** — их собирает фабрика `autoload/Look.gd`:

```gdscript
node.material = Look.mat(Color(...))          # тун + контур (alpha<1 → прозрачный)
node.material_override = Look.emissive(c, 3)  # тун со свечением
```

Параметры (uniforms), которые выставляет фабрика:
- `toon.gdshader` / `toon_alpha.gdshader`: `albedo`, `tex` (по умолч. белая),
  `bands` (ступеней света), `emission_color`/`emission_energy` (только у `toon`).
- `outline.gdshader`: `width` (толщина, м), `color` (цвет контура).

## Подключить текстуру к модели

В импорт-настройках `.glb` сделай материал «extractable», поставь на surface
`ShaderMaterial` с `toon.gdshader` и в параметр `tex` подставь текстуру модели.

## Зачем два шейдера (toon + toon_alpha)

Прозрачность в Godot нельзя включить «на лету»: шейдер, пишущий в `ALPHA`, всегда
идёт в прозрачную очередь рендера. Поэтому непрозрачные объекты держим на
`toon.gdshader` (быстрее, без артефактов сортировки), а прозрачные — на
`toon_alpha.gdshader`. Фабрика выбирает нужный по `alpha` цвета автоматически.
